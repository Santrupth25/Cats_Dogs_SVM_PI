# 🐱🐶 Cats vs Dogs Classification using SVM

This project implements a **Support Vector Machine (SVM)** model to classify images of **cats and dogs** using classical machine learning techniques.

---

## 📊 Dataset

- Source: Kaggle - Dogs vs Cats Dataset  
- Images are preprocessed by:
  - Converting to grayscale  
  - Resizing to 64x64  
  - Flattening into feature vectors  

---

## 🚀 Features

- Image preprocessing using OpenCV  
- SVM model training  
- Accuracy and classification report  
- Confusion matrix visualization  
- Sample prediction visualization  
- Accuracy graph  

---

## 🛠️ Tech Stack

- Python  
- OpenCV  
- NumPy  
- Scikit-learn  
- Matplotlib  

---

## 📈 Model Performance


Accuracy: 0.44


### Classification Report

| Class | Precision | Recall | F1-Score |
|------|----------|--------|----------|
| Cat  | 0.50     | 0.44   | 0.47     |
| Dog  | 0.39     | 0.44   | 0.42     |

## 🧠 Project Explanation

This project uses **Support Vector Machine (SVM)** for image classification.

Steps followed:
1. Load and preprocess images  
2. Convert images to grayscale  
3. Resize and flatten images  
4. Train SVM classifier  
5. Evaluate using accuracy and confusion matrix
